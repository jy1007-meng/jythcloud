自学 AI 一年，我的学习路线和资源分享

这篇文章想写很久了。大一的时候我对 AI 只有模糊的印象——ChatGPT 很厉害、自动驾驶很酷、学 AI 很难。一年后的今天，我虽然离"专家"还差得很远，但至少能动手做点东西了。这篇分享一下我自学 AI 的过程和用过的资源，希望能给同样想入门的同学一点参考。

24| 25|

我的背景

26|

我是湖北工程学院数据科学与大数据技术专业的在校生。大一基本是在上课考试中度过的，真正开始认真自学 AI 是从大二上学期开始的。自学之前，我有一些 Python 基础（学校课程学的），数学水平是高考数学 110+ 的水平——够用，但需要补。

27| 28|

第一阶段：Python + 基础（0-2 个月）

29|

虽然学校教过 Python，但应付 AI 相关的库（NumPy、Pandas、Matplotlib）远远不够。这个阶段我主要做了两件事：

30|
    31|
  • 刷了一遍 Pandas 官方文档的 10 minutes to pandas。 对 DataFrame 的基本操作有了概念。
    • 32|
  • 动手做小练习。 在 Kaggle 上找了一些入门级的数据集，试着读入、看看数据、画几张图。不求分析得多深入，只要跑通流程就行。
    • 33|
34|

这个阶段没必要买课，网上免费资源完全够了。Pandas 中文文档、菜鸟教程、B 站上找几个教学视频跟着敲一遍就行。

35| 36|

第二阶段：机器学习入门（2-5 个月）

37|

这个阶段是最容易放弃的。从"会用 Pandas 处理数据"到"理解机器学习算法"之间有一道不小的坎。

38|

我用的资源是吴恩达的 Machine Learning Coursera 课程。很多人推荐，确实讲得好——数学推导不多但概念讲得很清楚。我花了两个月跟完，每一节都做笔记，课后作业自己敲一遍。

39|

学完之后用 Scikit-learn 做了几个小项目：

40|
    41|
  • 用线性回归预测房价（波士顿房价数据集）
    • 42|
  • 用逻辑回归做鸢尾花分类
    • 43|
  • 用 K-Means 对用户消费数据进行聚类
    • 44|
45| 46|
47| 💡 我的体会：这个阶段最容易犯的错误是想一口气搞懂所有算法背后的数学原理。没必要。先理解每个算法是干什么的、输入输出是什么、什么时候用，然后调库跑起来。数学以后可以慢慢补，但如果在这个阶段因为数学太难过早放弃，就什么都没了。 48|
49| 50|

第三阶段：深度学习 & 大模型（5-8 个月）

51|

学完机器学习基础之后，我开始接触深度学习和 NLP。这个阶段的学习路径比较乱，因为大模型技术更新太快了：

52|
    53|
  • PyTorch 入门。 跟着官方教程的 60-minute blitz 敲了一遍，了解了张量、自动求导、模型定义和训练的基本流程。
    • 54|
  • Transformer 架构。 读了那篇著名的 "Attention Is All You Need" 论文——说实话第一次读只理解了大概 30%，后来看了几个图解视频才慢慢明白。
    • 55|
  • 大模型 API 调用。 前面一篇博客写了第一次调通大模型 API 的过程。这个阶段最兴奋的事情就是发现"原来我也可以调用这些强大的模型"。具体可以看我那篇大模型 API 调用入门。
    • 56|
57| 58|

第四阶段：做项目 & 输出（8-12 个月）

59|

到了这一步，我觉得应该从"学"转向"做"了。做了几个小项目：

60|
    61|
  • 基于大模型 API 的对话机器人（就是 API 调用入门那篇的内容）
    • 62|
  • 用 LangChain 搭建的自动查天气 Agent
    • 63|
  • 这个 JYTHCloud 团队网站（从服务器搭建到前端实现全流程）
    • 64|
65|

做项目的过程中，暴露了我在工程能力上的不足：不会写单元测试、代码组织混乱、部署上踩了很多坑。但反过来想，如果不去做项目，我根本不会意识到这些问题的存在。

想对刚入门的同学说

83|
    84|
  1. 不要等"准备好"再开始。 装好 Python、import 一个库、跑通一个例子，你就已经开始了。不需要先看完三本数学书。
    2. 85|
  2. 动手 > 看教程。 看十个小时的视频不如自己敲一小时的代码。
    3. 86|
  3. AI 没有那么高不可攀。 一年前的我觉得 AI 是大厂科学家才做的事。一年后的我知道，只要愿意花时间，普通在校生也能做点有意思的东西出来。
    4. 87|
  4. 接受"不知道"。 遇到不懂的概念很正常，先记下来，继续往前走。很多时候学到后面前面就自然懂了。
    5. 88|
89|

这一年学到了不少东西，但也只是刚入门而已。后面继续学、继续做、继续分享。希望能和更多同学一起交流。
